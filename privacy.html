Privacy Policy

How Zcode protects your data and respects your privacy

Introduction

Zcode is a macOS development tool created by Amir Hayek. We're committed to protecting your privacy while providing a fast, capable IDE for building iOS and macOS projects.

Your projects, source code, and local files remain completely under your control on your machine. We collect minimal, anonymous usage data only to improve Zcode's functionality and stability.

What We Don't Collect

Zcode respects your privacy. We explicitly do not collect:

  • Your source code — Never transmitted, never stored on our servers
  • Project files or contents — Your projects stay completely on your machine
  • Git history or data — Commits, branches, diffs, and all git information remain local
  • Build logs or error details — Build output and errors are not sent anywhere
  • Personal information — We don't ask for your name, email, or any identifiers
  • File system information — We don't scan or transmit details about your files

If data isn't listed in the "What We Do Collect" section below, we don't collect it.

What We Do Collect

Zcode uses Aptabase, an analytics service, to collect anonymous usage events. These events help us understand how developers use Zcode so we can improve the app.

Events We Track

  • App lifecycle: App launches
  • Build & Test: Build/test starts, completions, outcomes, durations, error/warning counts, number of targets
  • Navigation: Which sections of the app you visit
  • Git operations: Success/failure of pulls, pushes, fetches, branches, commits, stashes (not commit contents—just operation type and result)
  • Simulator management: Device selection and simulator launches
  • Settings changes: Feature toggles and preferences you enable/disable
  • Package manager: Dependency additions, removals, package resolutions
  • Archives: App archive creation and export attempts
  • Terminal usage: Command execution (command names only, no content)

What These Events DON'T Include

  • No project names
  • No file paths
  • No source code or output content
  • No personal identifiers (your name, email, etc.)
  • No IP addresses
  • No system details beyond basic platform info

Tracking Method

Each Zcode installation gets a random, anonymous device ID (UUID). No user account or login is required. The same device ID persists across sessions so we can measure feature adoption and stability, but it's never linked to you as a person.

Local Operations Stay Local

Zcode runs many operations entirely on your machine with zero network transmission:

  • Git operations (clone, pull, push, commit, branch switching, stashing)
  • Xcode build and test execution
  • Simulator management and app launching
  • Project scheme editing and build settings
  • Code signing and certificate management
  • Build history storage

All of these are handled locally. Nothing leaves your computer unless you explicitly push to a git remote.

Analytics & Tracking Details

About Aptabase

Aptabase is a privacy-focused analytics platform designed for app developers. It complies with GDPR, CCPA, and other privacy regulations. Aptabase does not use cookies, does not create user profiles, and does not share data with third parties.

Aptabase Privacy: https://aptabase.com/legal/privacy

No Third-Party Sharing

We do not share analytics data with anyone else. No advertisers, no data brokers, no marketing platforms. Only Aptabase (our analytics processor) sees this data.

GDPR & CCPA Compliance

Zcode and Aptabase comply with GDPR (EU General Data Protection Regulation) and CCPA (California Consumer Privacy Act). Since analytics are anonymous and not linked to personal identity, your data is already protected.

Data Retention & Deletion

Retention Policy

Aptabase retains analytics data for 1 year from the event date. After 1 year, data is automatically deleted.

Request Data Deletion

If you want your analytics data deleted immediately (before the 1-year deletion), contact:

Email: hayek_dev@icloud.com

Include "Privacy Request - Data Deletion" in the subject line. We'll process deletion requests within 30 days.

Your Privacy Rights

Under GDPR, CCPA, and other privacy laws, you have the right to:

  • Know what data is collected — See above
  • Access your data — Request a copy of any analytics data associated with your device ID
  • Delete your data — Request immediate deletion of analytics associated with your Zcode installation
  • Opt out of tracking — Disable analytics collection (see "How to Opt Out" below)

To exercise these rights, contact: hayek_dev@icloud.com

Include your request type in the subject line (e.g., "Privacy Request - Data Access"). We'll respond within 30 days.

How to Opt Out of Analytics

You can completely disable analytics collection in Zcode's Settings:

  1. Open Zcode Settings (Cmd+,)
  2. Click the General tab
  3. Find the "Send Analytics" toggle near the bottom
  4. Toggle OFF to disable analytics
  5. Toggle ON to re-enable

When analytics are disabled, no data is sent to Aptabase—not even anonymous device IDs or event names. Zcode works exactly the same with analytics off; you'll just miss out on helping us improve the app.

Summary

The bottom line: Your projects stay on your machine. We collect minimal, anonymous usage data via Aptabase to improve Zcode. You can opt out anytime via Settings. Your data is deleted after 1 year, and you can request deletion anytime.
