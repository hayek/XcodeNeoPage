Frequently Asked Questions

What is Xcode Neo?

Xcode Neo is a modern Mac app for building native iPhone, iPad, and Mac applications with advanced developer tools and AI integration.

Does Xcode Neo include a code editor?

No, source code is just an implementation detail.
Still want to edit? "Open in Xcode" buttons are there for the remaining few of you.

When will the beta end?

We're actively developing Xcode Neo and will announce the stable release timeline soon. Stay tuned for updates.

Can I use my own AI models?

Yes, Xcode Neo supports integrating your own AI models through the MCP (Model Context Protocol) system.

What platforms does Xcode Neo support?

Xcode Neo is a Mac app that allows you to build apps for iPhone, iPad, and macOS.

Do I need Xcode installed?

Yes, Xcode Neo requires a full Xcode installation to build, run, test, and manage your iOS and macOS projects. You can download Xcode for free from the App Store.
